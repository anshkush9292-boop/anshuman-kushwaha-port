## Lessons Learned
- Choosing a single-file implementation required abandoning framework-centric 3D libraries (Three.js/GSAP) and instead successfully leveraging advanced CSS animations paired with Vanilla JS to achieve a premium feel without the overhead.
- Keeping asset dependencies external via CSS/JS allows immediate preview, but user chose to defer base64 image generation for performance/workflow preferences.
- Heavy use of JSON-LD schemas embedded directly in the `<head>` sets a strong foundation for Answer Engine Optimization (AEO).
