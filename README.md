# Anshuman Kushwaha — AI-Driven Digital Growth Strategist

Welcome to my portfolio! This project showcases a high-performance, AI-augmented digital marketing portfolio designed for professional impact and search visibility.

## 🚀 Key Features

- **Modern Hero Section**: Features a custom animated SVG robot and premium glassmorphism AI Strategist card.
- **Marketing Tech Network**: An interactive skills visualization showing the interconnected nature of modern digital marketing tools.
- **AI-Augmented Content**: Built with a focus on AEO (Answer Engine Optimization) and search visibility for AI search engines like Perplexity and ChatGPT.
- **Performance Focused**: Minimal vanilla JS/CSS build for maximum speed and smooth animations.

## 🛠️ Technical Stack

- **Core**: HTML5, CSS3, Vanilla JavaScript
- **Design System**: Syne & DM Sans typography, Custom Orange & White theme
- **Animations**: CSS Keyframes, Intersection Observer for scroll reveals
- **Tools**: Google Ads, Meta Ads, YouTube SEO, GA4, Looker Studio, AI Automation

## 📁 Repository Structure

- `index.html`: Main production build containing the full portfolio.
- `_assets/`: (Planned) Directory for optimized images and documents.
- `scripts/`: Helper scripts for maintenance and deployment.

## 📈 Deployment

The portfolio is designed for static hosting (Vercel, GitHub Pages, etc.).
